# OpenPay OpenUSD (OUSD)

**OpenPay OpenUSD (OUSD)** is a USD-pegged utility stable token designed for payments, lending, staking, savings, merchant transactions, and governance within the OpenPay ecosystem.

Issued in collaboration with **Pi Network** and **MrWain Foundation**, OUSD aims to provide a stable digital currency with real-world utility.
